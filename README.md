- TypeScript v5

- husky v9
- eslint v8
- prettier v3

## Prepared to run server

### corepack

```bash
corepack enable
```

### Install Package modules

```bash
pnpm install
```

### Run Server

```bash
pnpm run <script_command> --filter <package_name>
```
